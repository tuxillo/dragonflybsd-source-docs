# HAMMER2 Chain Layer

!!! note "Documentation Status"
    This page documents the chain layer based on `hammer2_chain.c` and `hammer2.h`.

## Overview

The chain layer is HAMMER2's central abstraction for managing block references in memory. Every on-disk blockref is represented in memory as a chain structure.
